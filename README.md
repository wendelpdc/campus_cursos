No código acima, terá a classe Campus, contendo os seguintes atributos da classe: código, nome e endereço. 
Esses atributos são para definir o campus. 
Ex: 2, Jardins de Anita, Itapajé. 

Dentro da classe Campus, será criado os métodos "add_curso" e "listar_curso" (fora o método especial __init__, que é adicionado primeiramente para garantir a inicialização dos objetos).
